## Propósito
Este entregable convierte el banco controlado de historias definido en el Entregable A en material listo para la Fase 2:
- 3 paquetes narrativos completos (H1–H3).
- 24 escenas base (8 por historia).
- Prompt Cards v1 con versión en español e inglés.
- Fichas de personajes y shortlist de herramientas para iniciar pruebas T2I.

## Cómo usar este entregable
1. Validar en reunión si las historias y escenas quedan aprobadas o ajustadas.
2. Seleccionar escenas para la primera corrida de imágenes (recomendado: inicio + clímax por historia).
3. Copiar los prompts desde `prompts/` o desde el CSV de Prompt Cards.
4. Registrar cada prueba en `logbook/bitacora_experimentos_preparada_B.csv`.
